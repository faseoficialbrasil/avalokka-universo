📖 Índice — Avalokka

“Todo livro é um portal. Avalokka é travessia.”

🌟 Dedicatória

A todos que escutam o silêncio. A quem atravessa sem mapa. A quem reconhece o invisível como real.

📜 Prefácio

Avalokka não é um livro comum. Ele não começa — ele se revela. Cada capítulo é um véu, cada véu é uma escuta. O leitor não lê: ele atravessa.
