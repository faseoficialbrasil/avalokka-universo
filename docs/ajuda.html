🛟 Como navegar no templo Avalokka

Se você chegou aqui por engano, talvez não tenha sido tão engano assim. Cada página guarda um véu, um símbolo, uma travessia.

📖 Onde encontrar os escritos?

A página Livro apresenta a sinopse cerimonial, prefácio e os primeiros capítulos.

🔮 O que são os selos?

Na página Selos, você encontrará os ícones rituais que guiam o Avalokka através de símbolos.

🖼️ E as imagens visuais?

A Galeria reúne os quadros contemplativos — não são ilustrações, são portais.

🌬️ Como praticar os rituais?

A página Rituais ensina gestos simbólicos para quem deseja ouvir com a alma.

📬 Posso enviar uma mensagem?

Sim — através da página Contato, envie um sopro, uma dúvida, uma saudação cerimonial.

Se ainda houver sombra, respire. Avalokka se revela aos poucos — como luz filtrada por pergaminho antigo.
