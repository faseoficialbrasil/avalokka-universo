Portais Vivos de Avalokka

Travessias que respiram. Inscrições que se movem. Véus que escutam.

Enaileal é o guardião de todos os portais vivos. Ele não abre portas — ele as escuta. E só os que escutam com ele podem atravessar.

Portais Revelados

  • Véu de Candelestrae — portal vertical, feito de luz líquida. Só responde ao silêncio verdadeiro.
  • Fenda de Nengmenias — dobra temporal invertida. Só se abre para quem já esteve lá.
  • Porta de Bastardus — feita de renúncia. Só se atravessa ao abandonar o que se ama.
  • Espelho de Oxides — sem forma. Reflete o que você não quer ver.
  • Portal dos Sonhadores Extintos — cidade sem portas. Só se entra ao esquecer que se está entrando.

Portais Ocultos

Alguns portais ainda não foram revelados. Enaileal os conhece — mas não os nomeia. Eles se abrirão quando o próximo cântico for cantado.
